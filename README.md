# file_preview_demo

基于文章内容整理的 Flutter 文件预览示例，重点演示两条实现路线：

- 全能型插件：`universal_file_viewer`
- 自由组合型：`photo_view` + `mini_pdf_epub_viewer` + 本地文本 / CSV 渲染 + `open_filex` 兜底

## Demo 内容

- 预置本地示例文件：图片、Markdown、CSV、TXT
- 远程 PDF 预览示例
- 系统文件选择器接入
- Android 权限演示
- 同一份文件在两种预览策略之间切换
